
# SocialFlux AI

An AI-powered automation system that collects the latest digital marketing news, rewrites it in simple language, and publishes it automatically to a WordPress website using the WordPress REST API.

## Features

* Collects digital marketing news daily
* Rewrites content using AI
* SEO-friendly titles
* WordPress REST API integration
* Automatic draft/publish
* Daily scheduler
* Logging
* Image support

## Tech Stack

* Python
* FastAPI
* OpenAI API
* WordPress REST API
